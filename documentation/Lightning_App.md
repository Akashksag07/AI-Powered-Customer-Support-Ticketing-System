# Lightning App

## Project Name
AI-Powered Customer Support Ticketing System

## Overview
A custom Salesforce Lightning App was created to provide a centralized workspace for managing customer support cases.

## Features
- Easy navigation through the Salesforce interface.
- Access to Case records.
- Integration with Reports and Dashboards.
- User-friendly layout for support agents.

## Components Included
- Cases
- Reports
- Dashboards
- Home Page

## Benefits
- Simplifies daily support operations.
- Improves productivity.
- Provides quick access to customer support information.
