# Functional Scope

## Project Name
AI-Powered Customer Support Ticketing System

## Objective
To develop a Salesforce-based application that automates customer support operations by managing customer tickets efficiently.

## Functional Requirements
- Create customer support cases.
- Assign case priority (High, Medium, Low).
- Track case status (New, Working, Escalated, Closed).
- Automatically send email notifications using Salesforce Flow.
- Generate reports for monitoring support activities.
- Provide a Lightning App for easy navigation.

## Users
- Customers
- Support Agents
- Support Managers
- Salesforce Administrator

## Expected Outcome
The system streamlines customer support, reduces manual effort, improves response time, and enhances customer satisfaction through automation.
