# Reports

## Project Name
AI-Powered Customer Support Ticketing System

## Overview
Salesforce Reports are used to monitor and analyze customer support activities.

## Reports Created
### 1. Cases by Status
Displays the number of cases in each status:
- New
- Working
- Escalated
- Closed

### 2. Cases by Priority
Shows the distribution of cases by priority:
- High
- Medium
- Low

### 3. Case Owner Report
Lists cases assigned to each support agent.

## Benefits
- Monitor support team performance.
- Identify high-priority issues.
- Track pending and resolved cases.
- Support better decision-making through analytics.
