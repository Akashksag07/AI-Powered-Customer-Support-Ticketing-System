# AI-Powered Customer Support Ticketing System

## Project Overview
The AI-Powered Customer Support Ticketing System is a Salesforce-based application designed to streamline customer support operations. It helps organizations manage customer issues efficiently by creating, tracking, prioritizing, and resolving support tickets using Salesforce automation.

## Objectives
- Automate customer support processes.
- Improve ticket tracking and management.
- Send automated email notifications.
- Generate reports for support performance.
- Demonstrate Salesforce Administration and Flow Automation.

## Technologies Used
- Salesforce Developer Edition
- Salesforce Flow
- Lightning App
- Reports & Dashboards
- Email Alerts

## Features
- Case Management
- Record-Triggered Flow
- Automated Email Notifications
- Custom Reports
- Lightning App Navigation
- User-Friendly Interface

## Project Workflow
1. Customer creates a support case.
2. Case is assigned and categorized.
3. Salesforce Flow automates the business process.
4. Email notification is sent automatically.
5. Support agent resolves the case.
6. Reports track support performance.
